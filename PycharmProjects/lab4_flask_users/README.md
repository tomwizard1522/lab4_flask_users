# Лабораторная работа №4: Управление пользователями

Веб-приложение на Flask для управления учетными записями пользователей с использованием базы данных SQLite.

## Функциональность

### Управление пользователями
- Просмотр списка пользователей (доступно всем)
- Просмотр детальной информации о пользователе (доступно всем)
- Создание новых пользователей (только авторизованным)
- Редактирование пользователей (только авторизованным)
- Удаление пользователей (только авторизованным)

### Аутентификация
- Вход в систему с проверкой логина и пароля
- Защита маршрутов с помощью Flask-Login
- Смена пароля для текущего пользователя

### Валидация данных
- Проверка логина (только латиница и цифры, минимум 5 символов)
- Проверка пароля (сложные требования безопасности)
- Проверка обязательных полей
- Подсветка ошибок в формах

### База данных
- SQLite с SQLAlchemy ORM
- Таблицы: пользователи, роли
- Автоматическая дата создания записей
